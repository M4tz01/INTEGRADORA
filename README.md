# INTEGRADORA
<img width="602" height="425" alt="image" src="https://github.com/user-attachments/assets/7011c597-3852-488f-8575-f71a1092acc0" />



En este proyecto se aplicó herencia usando una clase padre llamada `Empleado` y las clases hijas `Medico` y `Administrativo`, las cuales heredan atributos y métodos. También se utilizó encapsulamiento colocando los atributos como `private` para proteger los datos y usando getters y setters. El polimorfismo se aplicó mediante `ArrayList<Empleado>`, permitiendo guardar diferentes tipos de empleados en una misma lista y usar el método `calcularPago()` de forma distinta en cada clase. Además, se usaron excepciones con `try-catch` para controlar errores cuando el usuario ingresa datos incorrectos. Se realizaron conversiones usando `Integer.parseInt()` y `Double.parseDouble()` para transformar texto en números. Finalmente, se aplicaron validaciones para evitar campos vacíos, valores negativos, cédulas duplicadas, teléfonos inválidos y correos incorrectos.
